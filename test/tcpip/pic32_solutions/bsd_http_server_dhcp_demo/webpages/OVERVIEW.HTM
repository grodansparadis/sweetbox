You are watching a real-time web server application powered by the Microchip PIC32MX™ microcontroller running the Microchip BSD TCP/IP Stack. Everything you see is served through a Microchip ENC28J60 10Base-T Ethernet controller using an SD memory card with FAT16.

The System Status web interface in the lower left portion of this page illustrates real-time status updates of the two LEDS (D10 and D9) and LCD text (if LCD module is present) status. This section is refreshed with the system information periodically using JavaScript.

The System Control web interface in the lower right portion of this page illustrates real-time control of the LED and LCD parameters. Toggling the LEDs are as simple as clicking the "Toggle" button. While displaying messages on the LCD is as easy as typing in the text and clicking "Submit Query".
